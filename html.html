Introduccion

HTML (HyperText Markup Language) es un lenguaje fundamental para la creación de sitios web. Con sus etiquetas y estructura jerárquica, HTML permite definir la apariencia y el contenido de una página en línea. Cada etiqueta desempeña un papel específico, desde encabezados y párrafos hasta imágenes y enlaces. Mediante el uso de atributos, se puede personalizar aún más el estilo y el comportamiento de los elementos. La semántica de HTML asegura una mejor accesibilidad y facilita la indexación por parte de los motores de búsqueda. Con HTML como base, los diseñadores y desarrolladores web pueden dar vida a sus ideas y brindar experiencias interactivas a los usuarios en todo el mundo.


Basico

Primero veremos lo mas fundamental de html.
Para empezar a escribir en este lenguaje primero tenemos que avisar al programa que vamos a escribir en HTML, eso lo hacemos con "!DOCTYPE", una etiqueta, antes de continuar tienen que saber que todos los codigos o como se conocen "etiquetas" empiezan usando el signo menor que y se cierra con el mayor que, pero para cerrar una etiqueta se usa esa misma etiqueta pero con un "/" al lado derecho del menor que, aunque hay algunas exepciones los cuales se usan sin cerrarlo como el "!DOCTYPE". despues de eso ponemos la etiqueta "head" y despues de ese va el body y estos dos van dentro de la etiqueta html.


etiquetas

etiqueta Definicion
p Esta etiqueta sirve para agregar un parrafo al documento html
a define un hipervinculo, naturalmente va con el atributo "href"
img con el podemos poner una imagen en la pagina
h1 - h6 Estos son para crear titulos subtitulos y cada una con un menor tamaño
br Este es para hacer saltos de linea en la pagina
!DOCTYPE con este definimos que el documento es html
html este es el que va a tener todo el codigo
head con este definimos la cabeza o la pestaña que muestra el navegador
body este tendra todo lo que se va a mostrar en la pagina
div con este hacemos diviciones en la pagina para tener bloques en siertas posiciones de la pagina segun como la queramos

Estas serian las etiquetas mas basicas que se usan a la hora de crear una pagina, pero hay una cantidad aun mayor de etiquetas que cubren diferentes necesidades que tengamos a la hora de crear el sitio, como etiquetas que cambian el tipo de letra que se usa, etiquetas para crear tablas como la tabla con las etiquetas, tambien hay etiquetas usadas para diseñar formularios, etc.
