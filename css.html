INTRODUCCION

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para dar estilo y mejorar la apariencia de las páginas web. Con CSS, se pueden controlar colores, fuentes, márgenes, tamaños y posicionamiento de elementos HTML. Proporciona una forma eficiente de separar la estructura de una página web de su presentación visual, lo que permite una fácil modificación y mantenimiento del sitio. CSS utiliza selectores para apuntar a elementos específicos y aplicar reglas de estilo. También ofrece características avanzadas como animaciones, transformaciones y diseño responsivo, lo que permite crear diseños modernos y atractivos. Es una herramienta fundamental para los desarrolladores web que desean personalizar y embellecer sus sitios con flexibilidad y control precisos.


ESTILOS

Ahora te mostraremos lo que puedes alterar con CSS:

  • Color de texto y fondo: permite establecer el color del texto y el fondo de un elemento.
  • Tipografía: permite especificar la fuente, tamaño, estilo y espaciado del texto.
  • Margen y relleno: controla el espacio alrededor y dentro de un elemento.
  • Bordes: permite establecer el estilo, grosor y color de los bordes de un elemento.
  • Posicionamiento: permite controlar la ubicación y el flujo de los elementos en la página.
  • Dimensiones: especifica el ancho, alto y tamaño de un elemento.
  • Alineación de texto: controla la alineación horizontal y vertical del texto.
  • Listas: permite personalizar el estilo de las listas numeradas y con viñetas.
  • Imágenes de fondo: permite establecer una imagen como fondo de un elemento.
  • Opacidad: controla la transparencia de un elemento y su contenido.
  • Animaciones y transiciones: permite crear efectos de animación y transición en los elementos.
  • Diseño responsivo: permite adaptar el diseño de la página a diferentes tamaños de pantalla.

Como agregar estilos

Te presentamos la sintaxis para gregar colores a un documento html, primero se ubica la etiqueta o elemento que se quiera modificar, se abre { } y en su interior escribimos la propiedad y el valor que vamos a modificar. 

            h1{   <------------Etiqueta o elemento.
                color: red; <-------- Propiedad y valor.
}

            p{   <------------Etiqueta o elemento.
