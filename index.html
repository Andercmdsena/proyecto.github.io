Introduccion


Bienvenidos a nuestro sitio web en donde les mostraremos lo que hemos aprendido sobre HTML, CSS y JavaScript. Abajo y en las demas paginas les enseñaremos lo basico sobre estos 3 lenguajes, los cuales se pueden codificar y ver usando frameworks o IDE, ademas, tenemos unos ejercicios donde podran pobrar sus conocimientos obtenidos en este pequeño proyecto, esperamos que disfruten y les guste nuestro sitio.
A continuacion, les presentamos un resumen de cada lenguaje, de ahi podran continuar a la pagina con el lenguaje que qieran empezar.


HTML es un lenguaje de marcado utilizado para crear y estructurar contenido web. Permite definir la estructura de una página, incluir imágenes, enlaces, videos y otros elementos. Es fundamental para el desarrollo de sitios web y se compone de etiquetas que indican la función de cada elemento en el documento.


CSS es un lenguaje de estilo utilizado para darle diseño y presentación a las páginas web creadas con HTML. Permite controlar colores, fuentes, tamaños, márgenes y otros aspectos visuales. Es clave para personalizar la apariencia de un sitio web y se aplica mediante reglas que seleccionan elementos HTML y les aplican estilos específicos.


JavaScript (JS) es un lenguaje de programación utilizado para agregar interactividad y funcionalidad dinámica a los sitios web. Permite manipular elementos HTML, responder a eventos del usuario, realizar cálculos, acceder a bases de datos y mucho más. Es esencial para crear aplicaciones web interactivas y se ejecuta en el navegador del cliente.
